# De fire prinsessene og Flamme

En magisk, digital bildebok om fire modige prinsesser og deres trofaste lille drage, Flamme.

Denne nettsiden er et interaktivt bildebok-konsept designet for å gi en sømløs og engasjerende leseopplevelse for barn. Den bruker moderne webteknologi (HTML, CSS og Vanilla JavaScript) for å skape en horisontal, boklignende navigasjon, komplett med nydelige glassmorphism-effekter og en adaptiv layout som ser fantastisk ut på både mobil, nettbrett og desktop.

## Innhold

- `index.html`: Den ferdiggenererte og formaterte bildeboken, klar for publisering.
- `source.html`: Originalkilden med den grunnleggende historien og strukturen.
- `convert_book.py`: En skreddersydd Python-generator som analyserer `source.html`, beregner optimal tekstlengde per side, og injiserer all moderne styling og interaktivitet for å kompilere det ferdige resultatet i `index.html`.
- `bilder/`: Mappen som inneholder alle illustrasjoner og karakterportretter til boken.

## Publisering

Boken er designet for å kunne hostes direkte og gratis via **GitHub Pages**. Sett kildegrenen til `main` eller `master`, og la GitHub ta seg av resten!
